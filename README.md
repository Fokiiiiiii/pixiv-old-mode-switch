# pixiv Old Mode Switch

pixivのタグ検索ページに  
**「すべて / 全年齢 / R-18」切り替えボタン**を追加するユーザースクリプトです。

最近のpixivではR-18表示の切り替えが少し分かりづらく、  
設定やフィルタを何度も開く必要があります。

このスクリプトを使うと、**昔のpixivのようにワンクリックで表示モードを切り替える**ことができます。

---

## できること

タグ検索ページに次のボタンを追加します。

[ すべて ]   [ 全年齢 ]   [ R-18 ]

クリックするだけで表示モードを簡単に切り替えることができます。

- **すべて**  
  R-18フィルタなし

- **全年齢**  
  全年齢作品のみ表示

- **R-18**  
  R-18作品を表示

---

## 表示位置

ボタンは **検索オプションの下** に追加されます。

検索オプション

[ すべて ]   [ 全年齢 ]   [ R-18 ]

---

## 必要なもの

このスクリプトは **Tampermonkey** で動作します。

Tampermonkey  
https://www.tampermonkey.net/

対応ブラウザ

- Chrome
- Edge
- Firefox
- Brave

---

## インストール方法

### 1. Tampermonkeyをインストール

ブラウザにTampermonkey拡張を追加します。

https://www.tampermonkey.net/

### 2. スクリプトを追加

このリポジトリのファイル

pixiv-mode-switch.user.js

をTampermonkeyに追加してください。

---

## 使い方

1. pixivのタグ検索ページを開きます

https://www.pixiv.net/tags/*/artworks

2. 検索オプションの下にボタンが表示されます

3. ボタンをクリックすると表示モードが切り替わります
